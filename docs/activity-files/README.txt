# Triple negative breast cancer data set - 10X Visium

Study:
Bassiouni et al. (2023) Spatial Transcriptomic Analysis of a Diverse Patient
Cohort Reveals a Conserved Architecture in Triple-Negative Breast Cancer.
(10.1158/0008-5472.CAN-22-2682)

Source:
https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE210616

DESCRIPTION:
The original data set results from triple-negative breast cancer (TNBC) tumor biopsies collected from a
diverse patient cohort. Patients underwent different therapeutic regimes. The tumor biopsies
were profiled with 10X Visium. The data set presented here is a subset of the original cohort.
Four samples have been selected for this example:
- sample_093D (Accession GSM6433590)
- sample_396A (Accession GSM6433617)
- sample_396C (Accession GSM6433618)
- sample_397D (Accession GSM6433623)
